# PaaS-TA 컨테이너 플랫폼

## 소개
네이티브 쿠버네티스 설치 가이드(Kubespray 설치, KubeEdge 설치) 및 쿠버네티스에 컨테이너 플랫폼을 배포하여 사용할 수 있는 방법(PaaS-TA를 통해 서비스 형태로 배포되는 CaaS 배포와 단독 배포)에 대한 설치 및 활용 가이드를 다루고 있다.
